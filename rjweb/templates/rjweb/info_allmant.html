{% blocktrans trimmed %} Resejämföraren är ett webbaserat verktyg som gör det möjligt att jämföra olika färdmedel (cykel, gång, kollektivtrafik och bil) med avseende på tid, sträcka, kostnad, miljöbelastning samt kaloriförbrukning. {% endblocktrans %}

{% blocktrans trimmed %} Resejämföraren ingår i ett större projekt som har målsättningen att öka användandet av hållbara transportmedel i regionen. Utgångspunkten är att de som omprövar sina vanor vill fatta beslut utifrån parametrar som är betydelsefulla i vardagen. Syftet med Resejämföraren är att tillhandahålla ett lättillgängligt verktyg som stödjer omprövandet av resvanor generellt och framhåller de hållbara färdsätten specifikt. {% endblocktrans %}

{% blocktrans trimmed %} Resejämförarens anvisning av sträcka är baserade på närmsta vägen från punkt A till punkt B för gång och cykel men snabbaste vägen för bilen. För gång- och cykelvägen innebär detta att den trafiksäkraste inte alltid anges. Tider för kollektivtrafik utgår från måndagar kl 8 medan tiderna för bil, cykel, och gång utgår från tider utanför rusningstimmarna. {% endblocktrans %}

{% blocktrans trimmed %} Tiderna som anges inkluderar tid för parkering, betalning av parkeringsavgifter samt gångväg till och från parkeringsplats. {% endblocktrans %}

{% blocktrans trimmed %} I Resejämföraren möjliggörs en jämförelse mellan cykel, gång, kollektivtrafik och bil för samtliga orter och adresser i Skåne och Köpenhamnsregionen. {% endblocktrans %}

{% blocktrans trimmed %} Årsvärdena bygger på tur och returresor. {% endblocktrans %}
