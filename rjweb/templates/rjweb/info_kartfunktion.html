{% blocktrans trimmed %} Resejämföraren bygger på en routing service som använder data från OpenStreetMap, vilket är en redigerbar kartfunktion som stäcker sig över hela världen. {% endblocktrans %}

{% blocktrans trimmed %} OpenStreetMap är en fri karttjänst som startades som ett ideellt projekt i syftet att ta fram geografisk information (främst vägkartor) för fri spridning. {% endblocktrans %}

{% blocktrans trimmed %} Kartfunktionen bygger på att användare registrerar sig kostnadsfritt på webbplatsen och med hjälp av GPS data lägger in sina spårloggar för att redigera kartan. {% endblocktrans %}
